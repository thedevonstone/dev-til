## aspect-ratio

The `aspect-ratio` property replaces the old padding-top percentage hack.

```css
/* Old hack */
.video-wrapper { padding-top: 56.25%; position: relative; }

/* Modern */
.video-wrapper { aspect-ratio: 16 / 9; }
.square { aspect-ratio: 1; }
```

## text-wrap: balance

Prevents awkward single-word last lines in headings.

```css
h1, h2, h3 {
  text-wrap: balance;
}
```

Browser redistributes line breaks so all lines are roughly equal length. Only applies up to ~6 lines for performance reasons.
