## git log graph view

A compact visual of branch history directly in the terminal.

```bash
git log --oneline --graph --all --decorate
```

Worth aliasing: `git config --global alias.lg "log --oneline --graph --all --decorate"`

## git tag for releases

Annotated tags store extra metadata (tagger, date, message) and are the right choice for releases.

```bash
git tag -a v1.2.0 -m "Release 1.2.0"
git push origin v1.2.0
git push origin --tags  # push all tags
```

Lightweight tags (no `-a`) are just pointers — fine for local bookmarks, not releases.
