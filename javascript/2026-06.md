## Array spread vs Array.from

Both convert iterables to arrays, but `Array.from` takes an optional map function.

```js
// spread
const arr = [...nodeList];

// Array.from with map — one step instead of two
const values = Array.from(nodeList, node => node.textContent);
```

`Array.from` also works on array-like objects (e.g. `arguments`) that spread doesn't handle.

## queueMicrotask for async without promises

`queueMicrotask` schedules a callback in the microtask queue — runs before the next task but after the current synchronous code.

```js
queueMicrotask(() => {
  // runs before setTimeout, after current call stack
  processQueue();
});
```

Useful when you need to defer work without the overhead of a full Promise.
