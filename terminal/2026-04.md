# Terminal — April 2026

## Shell history tricks

A few that I keep forgetting and then rediscovering:

```bash
!!         # repeat last command
!$         # last argument of last command
Ctrl+R     # reverse search through history
Ctrl+G     # cancel the reverse search
Alt+.      # paste last argument (press repeatedly to cycle)
```

`Alt+.` is the one I use most — saves retyping long paths.

## xargs for batch operations

When you want to run a command on each line of output:

```bash
# delete all .log files older than 7 days
find . -name "*.log" -mtime +7 | xargs rm

# run a command on each result with a placeholder
find . -name "*.json" | xargs -I{} node validate.js {}
```

The `-I{}` flag lets you control where the input goes in the command.

## Process substitution

Pass the output of a command as a file argument without a temp file:

```bash
diff <(git show HEAD:file.txt) <(cat file.txt)
```

`<(command)` creates a virtual file descriptor. Works anywhere a filename
is expected.
