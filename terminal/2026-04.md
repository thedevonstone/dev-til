## Shell history tricks

A few that I keep forgetting and then rediscovering:

```bash
!!         # repeat last command
!$         # last argument of last command
Ctrl+R     # reverse search through history
Ctrl+G     # cancel the reverse search
Alt+.      # paste last argument (press repeatedly to cycle)
```

`Alt+.` is the one I use most — saves retyping long paths.

## xargs for batch operations

When you want to run a command on each line of output:

```bash
# delete all .log files older than 7 days
find . -name "*.log" -mtime +7 | xargs rm

# run a command on each result with a placeholder
find . -name "*.json" | xargs -I{} node validate.js {}
```

The `-I{}` flag lets you control where the input goes in the command.

## Process substitution

Pass the output of a command as a file argument without a temp file:

```bash
diff <(git show HEAD:file.txt) <(cat file.txt)
```

`<(command)` creates a virtual file descriptor. Works anywhere a filename
is expected.

## fzf — fuzzy finder for everything

`fzf` is a command-line fuzzy finder that integrates with almost everything.

```bash
# Interactive file selection
vim $(fzf)

# Fuzzy search through history (Ctrl+R replacement)
history | fzf

# Kill a process interactively
kill $(ps aux | fzf | awk '{print $2}')
```

Install: `brew install fzf` then run `$(brew --prefix)/opt/fzf/install` for shell integration.

## direnv for per-directory environment variables

`direnv` loads environment variables from a `.envrc` file when you `cd` into a directory.

```bash
# .envrc
export DATABASE_URL=postgres://localhost/myapp_dev
export API_KEY=dev-key-here

# activate
direnv allow
```

Automatically unloads when you leave the directory. No more manually sourcing `.env` files. Install: `brew install direnv`

## Process substitution

Use the output of a command as if it were a file — without a temp file.

```bash
diff <(sort file1.txt) <(sort file2.txt)
comm <(git ls-files) <(find . -name "*.js" | sed 's|./||')
```

`<(command)` creates a virtual file descriptor. Works anywhere a filename is expected.
