## git shortlog for contribution summary

Summarizes commits by author — useful for changelogs or seeing who worked on what.

```bash
git shortlog -sn           # count commits per author, sorted
git shortlog -sn v1.0..HEAD  # since a tag
```

## git config aliases worth setting

A few aliases that save keystrokes every day.

```bash
git config --global alias.st   "status -s"
git config --global alias.co   "checkout"
git config --global alias.br   "branch -a"
git config --global alias.lg   "log --oneline --graph --decorate --all"
git config --global alias.undo "reset HEAD~1 --mixed"
```
