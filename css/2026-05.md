## CSS nesting

Native CSS nesting is now supported without a preprocessor.

```css
.card {
  padding: 1rem;

  & h2 {
    font-size: 1.25rem;
  }

  &:hover {
    background: var(--hover-bg);
  }
}
```

The `&` is required (unlike Sass). Supported in all modern browsers.

## overscroll-behavior

Controls what happens when you scroll to the edge of an element.

```css
/* Prevent the page from scrolling when a modal's content hits the bottom */
.modal-body {
  overflow-y: auto;
  overscroll-behavior: contain;
}
```

`contain` stops scroll chaining. `none` also disables the pull-to-refresh bounce effect.
